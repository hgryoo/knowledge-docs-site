## CUBRID

CUBRID 코드베이스를 소스 레벨에서 읽는다 — 스토리지, MVCC, 락 매니저, 복구,
질의 파이프라인, 복제/HA, 그리고 PL 계열.

**먼저 읽기:** [아키텍처 개요](cubrid/cubrid-architecture-overview/)

## PostgreSQL

PostgreSQL 백엔드(REL_18)를 모듈 단위로 읽는다 — 공유 메모리 프로세스 모델,
WAL 내구성 축, 스토리지와 플러거블 액세스 메서드, 플래너/익스큐터 파이프라인,
카탈로그/캐시 계층, 복제, 확장성 표면, 그리고 버전 진화 문서까지.

**먼저 읽기:** [아키텍처 개요](postgres/postgres-architecture-overview/)
